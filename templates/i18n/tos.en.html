This application is using several technologies to run this service :
  • AppEngine To run the WebSite
  • Android : The Operating System used by your phone
  • Gloud device messaging (C2DM) : To directly send notification from the website to your phone, without running the software continuously (save battery)
  • Channel API : To enable your phone to send data in real time to your browser
  • Icons from icons-land : Not really a technology but needed for licence reasons
