Cette application utilise diverse technologies :
  • AppEngine pour faire fonctionner le site web
  • Android : Le système d'exploitation de votre téléphone
  • Google Cloud Messaging (GCM) : Pour envoyer directement des notifications depuis ce site web, sans avoir à laisser tourner l'application en tache de fond (ainsi la batterie a une plus longue durée de vie)
  • Channel API : Pour permettre à votre téléphone de vous répondre immédiatement
  • Icons from icons-land : Ce n'est pas une technologie, mais c'est pour des raisons de licences
