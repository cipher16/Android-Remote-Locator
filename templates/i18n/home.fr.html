Le but d'Android Remote Locator, est de vous permettre depuis ce site web de faire plusieurs actions à distances.

Les fonctionnalités

  • Connaitre la geolocalisation de votre android
  • De pouvoir le bloquer en cas de vol
  • De le faire sonner en cas de perte (sous les coussins du canapé après une soirée arrosée ;))

A ne pas OUBLIER!!

L'application est en cours de développement, et est susceptible de voir son développement arreté, ou son fonctionnement arreté.

Code source disponible

Son code source est disponible sur gitHub

  • L'application android (écrite en java)
